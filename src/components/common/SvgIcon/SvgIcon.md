Adds an SVG-icon from sprite created by 'vue-cli-plugin-svg-sprite' plugin.

___


#### Examples of the component
```jsx
<SvgIcon
  name="tick"
  width="30"
  height="30"
  title="tick" />

<SvgIcon
  name="tick"
  width="30"
  height="30"
  title="tick"
  class="dark-icon" />

<SvgIcon
  name="tick"
  width="30"
  height="30"
  title="tick"
  class="brand-icon" />

<SvgIcon
  name="tick"
  width="30"
  height="30"
  title="tick"
  class="gray-icon" />

<span style="background:gray;display:inline-block;padding:4px;">

<SvgIcon
  name="tick"
  width="30"
  height="30"
  title="tick"
  class="white-icon" />

</span>
```
